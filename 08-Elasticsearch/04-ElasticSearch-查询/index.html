04-ElasticSearch-查询

本文最后更新于：2019年3月13日

ElasticSearch查询

ElasticSearch支持的查询

query string search

GET /index_name/_search?q=name:czm&sort=age:desc

query DSL

Match_All

GET /index_name/_search
{
“query”: {
“match_all”: {}
}
}

Match

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT”
}
}
}

multi_fields

GET example_index/_search
{
“query”: {
“multi_match”: {
“query”: “查询的内容”,
“fields”: [“chineseName”,”englishName”]
}
}
}

自定义排序 –desc/asc

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT”
}
},
“sort”: [
{
“FIELD”: {
“order”: “desc”
}
}
]
}

分页查询

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT”
}
},
“from”: 0,
“size”: 20
}

只返回部分字段

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT”
}
},
“_source”: [“FIELD”,”FIELD”]
}

bool filter

  • filter仅仅作为过滤条件，对于搜索结果的权重没有贡献

GET /index_name/_search
{
“query”: {
“bool”: {
“must”: [
{
“match”: {
“FIELD”: “TEXT”
}
}
],
“should”: [
{
“match”: {
“FIELD”: “TEXT”
}
}
],
“filter”: {
“range”: {
“FIELD”: {
“gte”: 10,
“lte”: 20
}
}
}
}
}
}

constant_score fliter

GET example_index/_search
{
“query”: {
“constant_score”: {
“filter”: {
“range”: {
“FIELD”: {
“gte”: 10,
“lte”: 20
}
}
},
“boost”: 1.2
}
}
}

full text query

  • 全文检索的意思是，在多个搜索词的时候，使用TF/IDF等算法进行权重计算，返回携带一个/多个搜索词的结果

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT1 TEXT2”
}
}
}

phrase search

  • 短语搜索的意思是，整个短语连续出现的时候，才能作为结果返回

GET /index_name/_search
{
“query”: {
“match_phrase_prefix”: {
“FIELD”: “TEXT1 TEXT2”
}
}
}

highlight search

GET /index_name/_search
{
“query”: {
“match”: {
“FIELD”: “TEXT”
}
},
“highlight”: {
“fields”: {
“FIELD”:{}
}
}
}

range query

GET example_index/_search
{
“query”: {
“range”: {
“FIELD”: {
“gte”: 10,
“lte”: 20
}
}
}
}

term query

  • 注意 term query不是keywoed query, 而是将整个查询的字符串（不分词）拿到倒排索引中去查

GET example_index/_search
{
“query”: {
“term”: {
“FIELD”: {
“value”: “VALUE”
}
}
}
}

terms query

GET example_index/_search
{
“query”: {
“terms”: {
“FIELD”: [
“VALUE1”,
“VALUE2”
]
}
}
}

搜索结果

返回结果 解释 例子
took 耗费时间(ms) 6
timed_out 是否超时 false
_shard.total shard总数 6
_shard.successful 成功的shard数 6
_shard.failed 失败的shard数 0
hits.total 能够匹配的条数 10
hits.max_score 最高分数 1
hits.hits 匹配结果

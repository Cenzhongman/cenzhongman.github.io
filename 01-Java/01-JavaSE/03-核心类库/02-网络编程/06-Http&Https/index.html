Http&Https

一、Http

影响一个 HTTP 网络请求的因素主要有两个：带宽和延迟。

  • 安全：
  • 带宽：增加缓存机制
  • 延迟
    • 浏览器阻塞（HOL blocking）：浏览器请求数量限制
    • DNS 查询（DNS Lookup）：浏览器DNS缓存
    • 建立连接（Initial connection）：减少频繁的三次握手

Http1.x

  1. HTTP0.9：是HTTP协议的第一个版本，只允许发送get请求，并且不支持请求头。一次请求对应一次响应。是短连接。
  2. HTTP1.0：
    1. 请求与响应支持头域（支持请求头）
    2. 响应对象以一个响应状态行开始
    3. 响应对象不只局限于超文本
    4. 开始支持客户端通过post方法向web服务器提交数据，支持GET、HEAD、POST方法
    5. 支持长连接（但是默认还是使用短连接）
  3. Http1.1
    1. 缓存
    2. 请求头增加Host
    3. keeplive长链接(减少不必要的三次握手)
  4. Http2.0
    1. 多路复用，会将传输的信息分割成小的信息和帧，降低延迟，提高吞吐量。
    2. 可以终端TCP连接。

Parser接口

所有的解析器都要实现这个接口，有两种方案来实现文档内容解析

方式一：自动解析(自动判断类型，实现解析，读取内容)

简单，能够读取一般的文档并获得文档信息，但是可定制性差。

import java.io.File;

import org.apache.tika.Tika;

public class TikaDemo {
    public static void main(String[] args) throws Exception {
        File file = new File("C:\\Users\\34566\\OneDrive\\docs\\Hadoop.The.Definitive.Guide.4th.Edition.2015.3.pdf");//
        // 创建tika实例
        Tika tika = new Tika();

        // 使用解析器解析
        String filecontent = tika.parseToString(file);
        System.out.println("文件内容是：" + filecontent);
    }
}

tika.parseToString(file);方法的实际流程是这样的

方式二：Parser接口的parse()方法(其实parseToString用的也是这样的方法，详情见源码)

方法参数定义：

parse(InputStream stream, ContentHandler handler, Metadata metadata, 
ParseContext context)

要使用这个方法，必须实例化任何实现这个接口的类(内容太多不一一列举)

值得一提的是

  • AutoDetectParser类提供了自动类型判断的解析器接口，自动判断类型之后进行调用
  • CompositeParser类，可以选择几个解析器作为一组解析器

实例分析：

1.实例化实现解析器接口的类

Parser parser = new AutoDetectParser();
//Parser parser = new CompositeParser();

2.实例化一个处理类的对象

Handler 描述

BodyContentHandler 这个类采用XHTML输出的主体部分，并写入该内容到输出写入或输出流。然后重定向XHTML内容到另一个内容处理程序实例。
LinkContentHandler 这个类检测，并挑选XHTML文档的所有H-参考标签和转发那些使用类似网络爬虫工具
TeeContentHandler 这个类可以帮助在同时使用多个工具

BodyContentHandler handler = new BodyContentHandler();

3.创建的元数据对象

Metadata metadata = new Metadata();

4.创建输入流

File file=new File(filepath)
FileInputStream inputstream=new FileInputStream(file);

// 或者
InputStream stream = TikaInputStream.get(new File(filename));

5.创建上下文对象

ParseContext context =new ParseContext();

6.实例化解析器对象，调用parse方法，并通过所有需要的对象

parser.parse(inputstream, handler, metadata, context);

完整示例：

import org.apache.tika.Tika;
import org.apache.tika.metadata.Metadata;
import org.apache.tika.parser.AutoDetectParser;
import org.apache.tika.parser.ParseContext;
import org.apache.tika.parser.Parser;
import org.apache.tika.sax.BodyContentHandler;

import java.io.File;
import java.io.FileInputStream;


public class TikaDemo {
    public static void main(String[] args) throws Exception {
        File file = new File("D:\\Desktop\\test.txt");//
        // 创建tika实例
        Tika tika = new Tika();

        Parser parser = new AutoDetectParser();

        BodyContentHandler handler = new BodyContentHandler();
        Metadata metadata = new Metadata();
        FileInputStream inputStream = new FileInputStream(file);
        ParseContext context = new ParseContext();

        parser.parse(inputStream, handler, metadata, context);
        System.out.println("内容是：" + handler.toString());
    }
}

Metadata

这个类实现了各种接口来支持不同的数据模型，依据的是Adobe公司创建的XMP标准。

get metadata

// 方法一：
// 接上文代码，必须在解析之后，原理是：传递一个空的元数据对象作为一个参数进入解析器，将会将元数据信息封装在metedata对象中
String[] metadataNames = metadata.names();

for(String name : metadataNames) {                
    System.out.println(name + ": " + metadata.get(name));
}

// 方法二：
// 存储的是Arrays的样式，也可以这样输出
System.out.println(Arrays.toString(metadata.names()));

add metadata

// 直接使用ky值就可以存入metaData，但是最好还是要符合XMP标准的
// XMP标准可以参考源码Metadata类的常量，以及各种接口中的类变量
metadata.add("Key","Value");

Language Identifier类

用于翻译

语言检测，针对没有语言元数据的文档，做必要的语言检测

检测原理：与日常于语料库进行匹配，Tika使用3-grams算法，支持的列表如下

LanguageIdentifier object=new LanguageIdentifier(“this is english”);

读取文档语言实例：

// 必须在解析器之后
import org.apache.tika.Tika;
import org.apache.tika.language.LanguageIdentifier;
import org.apache.tika.metadata.Metadata;
import org.apache.tika.parser.AutoDetectParser;
import org.apache.tika.parser.ParseContext;
import org.apache.tika.parser.Parser;
import org.apache.tika.sax.BodyContentHandler;

import java.io.File;
import java.io.FileInputStream;
import java.util.Date;


public class TikaDemo {
    public static void main(String[] args) throws Exception {
        File file = new File("D:\\Desktop\\test.txt");//
        // 创建tika实例
        Tika tika = new Tika();

        // parser
        Parser parser = new AutoDetectParser();
        BodyContentHandler handler = new BodyContentHandler();
        Metadata metadata = new Metadata();
        FileInputStream inputStream = new FileInputStream(file);
        ParseContext context = new ParseContext();
        parser.parse(inputStream, handler, metadata, context);

        // get language
        LanguageIdentifier object = new LanguageIdentifier(handler.toString());
        System.out.println("Language name :" + object.getLanguage());
    }
}
